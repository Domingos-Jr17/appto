# Architect Command

Invokes the **architect** agent for system design decisions.

## Usage

```
/architect [question or problem]
```

## What It Does

1. Analyzes the current architecture
2. Identifies the design challenge
3. Proposes multiple options with trade-offs
4. Recommends the best fit for the project

## Example

```
/architect Should we use event-driven or request-response for the notification system?
```
