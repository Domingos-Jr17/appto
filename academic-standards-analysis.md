## 5. Recomendações para Melhoria do Sistema

### 5.1 Para Trabalhos Escolares (SECONDARY)

A estrutura actual (Introdução → Desenvolvimento → Conclusão) é **adequada** para o ensino secundário, mas precisa de:

1. **Adicionar citações indirectas** — mesmo no secundário, deve-se citar fontes
2. **Adicionar abstract em inglês**
3. **Melhorar o Desenvolvimento** para ter subtópicos mais claros
4. **Adicionar secção de Referências** com fontes reais

### 5.2 Para Trabalhos Universitários (HIGHER_EDUCATION)

A estrutura actual é **insuficiente**. Precisa de:

1. **Estrutura completa em capítulos:**
   - Capa + Página de Rosto
   - Resumo + Abstract
   - Índice
   - 1. Introdução (com objectivos, justificativa, metodologia)
   - 2. Revisão da Literatura (com citações de autores)
   - 3. Materiais e Métodos
   - 4. Resultados
   - 5. Discussão
   - 6. Conclusão e Recomendações
   - Referências Bibliográficas

2. **Citações no texto** (ABNT autor-data)
3. **Referências reais** formatadas conforme ABNT NBR 6023
4. **Metodologia de pesquisa** descrita
5. **Dados empíricos** ou análise documental

### 5.3 Prompt de Geração — Melhorias Necessárias

O prompt actual pede apenas "Introdução, Desenvolvimento, Conclusão". Para trabalhos universitários, deve pedir:

```
Gere um trabalho académico com a seguinte estrutura:

1. Introdução (400-500 palavras)
   - Contextualização do tema
   - Justificativa
   - Objectivos (geral e específicos)
   - Metodologia
   - Estrutura do trabalho

2. Revisão da Literatura (800-1000 palavras)
   - Estado da arte
   - Principais teorias e autores
   - Lacunas na literatura
   - Citações no formato (SOBRENOME, ano)

3. Materiais e Métodos (300-400 palavras)
   - Tipo de pesquisa
   - Procedimentos
   - Análise de dados

4. Resultados e Discussão (800-1000 palavras)
   - Apresentação dos achados
   - Comparação com a literatura
   - Implicações

5. Conclusão (300-400 palavras)
   - Sumário dos resultados
   - Contribuições
   - Limitações
   - Recomendações

Incluir pelo menos 8-12 citações de autores reais no formato ABNT.
```

## 6. Conclusão da Análise

O conteúdo gerado é **textualmente bem escrito** e **contextualmente relevante** para Moçambique. No entanto, **não constitui um trabalho de pesquisa académico válido** segundo as normas da UEM porque:

1. **Falta o núcleo científico:** sem metodologia, resultados, discussão ou citações
2. **Estrutura simplificada:** adequada para escola secundária, não para universidade
3. **Sem fundamentação teórica:** nenhum autor ou teoria citada

**Score de conformidade académica: 35/100**

O sistema funciona bem como **assistente de escrita** para gerar conteúdo base, mas precisa de ser expandido para incluir a estrutura completa de pesquisa académica quando o nível educacional for HIGHER_EDUCATION.
