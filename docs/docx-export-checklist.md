# DOCX Export Checklist

Use este checklist sempre que houver mudança na geração ou exportação de trabalhos académicos.

1. Gerar um `.docx` real para `SECONDARY_WORK`, `TECHNICAL_WORK` e `HIGHER_EDUCATION_WORK`.
2. Abrir o ficheiro no Microsoft Word.
3. Actualizar os campos do documento para forçar o preenchimento do sumário automático.
4. Confirmar que o sumário lista apenas headings do corpo, sem `Capa`, `Folha de Rosto` ou `Índice`.
5. Confirmar que a capa está coerente com o nível de ensino:
   `SECONDARY`: capa escolar simples
   `TECHNICAL`: capa disciplinar/técnica
   `HIGHER_EDUCATION`: capa universitária e folha de rosto quando aplicável
6. Confirmar que o tipo de trabalho aparece humanizado, nunca como `SECONDARY_WORK`, `TECHNICAL_WORK` ou `HIGHER_EDUCATION_WORK`.
7. Confirmar que não existe HTML/CSS literal no documento final.
8. Confirmar que não existem headings duplicados no início das secções.
9. Confirmar que a secção `Referências` aparece:
   preenchida com entradas legíveis, ou
   com aviso explícito de revisão manual quando a busca automática não foi suficiente.
